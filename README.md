# Learning ASP.NET

Welcome to the ASP.NET learning guide! This document provides an overview and resources to help you get started with ASP.NET development.

## What is ASP.NET?

ASP.NET is an open-source, server-side web application framework developed by Microsoft. It enables developers to build dynamic web sites, applications, and services using .NET.

## Key Features

- **Cross-platform**: Runs on Windows, Linux, and macOS.
- **High performance**: Optimized for speed and scalability.
- **MVC architecture**: Supports Model-View-Controller pattern.
- **Security**: Built-in authentication and authorization.
- **Rich ecosystem**: Large library support and active community.

## Getting Started

1. **Install .NET SDK**  
    Download and install the [.NET SDK](https://dotnet.microsoft.com/download).

2. **Create a New Project**  
    ```bash
    dotnet new webapp -n MyAspNetApp
    cd MyAspNetApp
    dotnet run
    ```

3. **Explore Tutorials**  
    - [ASP.NET Core Documentation](https://learn.microsoft.com/aspnet/core/)
    - [Microsoft Learn: ASP.NET Core](https://learn.microsoft.com/training/paths/build-web-apps-with-aspnet-core/)
